Name formating of units

Model is the generation number of model ie: iphone 5 is 5 gen iphone
unit is the unit number
extention is for futher description
	P: proto- is a prototype, or proof of concept
	D: development- is a development model is upgraded to become production model
	 : Nothing is a production model
	C: is a custom order model

Example:
R1-0-D
Model: R1
Unit: 0
Extention: D for Development
